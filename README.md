# ФинПро Frontend

ФинПро фронтенд.

## Версия

NodeJS > 20.7.0

## Установка и конфигурация

```shell
git clone git@git.f7m.ru:finpro/finpro-front.git
cd finpro-front
npm ci
```
## Запуск приложения

- development mode

```shell
npm start
```
- production mode

```shell
npm build
```

## Зависимости

См `package.json` в root директории.
